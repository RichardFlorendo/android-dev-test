📌 PokeAPI App - README
📖 Overview
The PokeAPI App is an Android application built using Kotlin & XML with View Binding, designed to fetch and display Pokémon data from the PokeAPI. Users can browse a paginated list of Pokémon, search for specific Pokémon by name, and view detailed stats, abilities, and images in a dedicated details screen.


📌 Features
✅ Pokémon Listing & Search – Fetches Pokémon data dynamically with pagination for seamless browsing.
✅ Detail View – Displays stats, abilities, height, weight, and images for each Pokémon.
✅ XML UI with View Binding – Uses traditional XML layouts with View Binding for improved efficiency.
✅ Coil Image Loading – Optimized image rendering for fast and smooth Pokémon sprite loading.
✅ Navigation Components – Seamless transitions between list and detail views.
✅ Error Handling – Displays user-friendly error messages when API calls fail.
✅ Future Offline Support – Plans to integrate Room Database for caching Pokémon data and allowing offline access.


🛠 Tech Stack
Programming Language: Kotlin
UI Framework: XML with View Binding
Networking: Retrofit
Image Loading: Coil
Architecture: MVVM (Model-View-ViewModel)
Navigation: Jetpack Navigation Component
